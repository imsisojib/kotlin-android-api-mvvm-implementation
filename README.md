# kotlin-android-api-mvvm-implementation
An app to show data from API Json file to Recyclerview using Kotlin Language in Android.

## Project Architecture
-Android `MVVM` Architecture

-Restful API

-Retrofit

-Recyclerview

-Kotlin Language
